## API Reference

### Countdown

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `to` | `Date` | required | Target date for the countdown |
| `showWhenExpired` | `boolean` | `false` | Continue showing time after expiration |
| `children` | `(state: CountdownState) => ReactNode` | required | Render function |

### CountdownState

The state object passed to the render function.

| Property | Type | Description |
|----------|------|-------------|
| `timeString` | `string` | Formatted time remaining (e.g., "2d 5h 30m") |
| `isExpired` | `boolean` | Whether the countdown has expired |
| `days` | `number` | Days remaining |
| `hours` | `number` | Hours remaining |
| `minutes` | `number` | Minutes remaining |
| `seconds` | `number` | Seconds remaining |
| `totalSeconds` | `number` | Total seconds remaining |
