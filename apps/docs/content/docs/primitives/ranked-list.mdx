## API Reference

### RankedList.Root

The root component that provides context for the ranked list.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `T[]` | required | Array of items to display |
| `getKey` | `(item: T) => string` | required | Function to get unique key for each item |
| `boundaries` | `number[]` | required | Indices where groups should be split |
| `labels` | `string[]` | required | Labels for each group |

### RankedList.Group

Container for a group of ranked items.

### RankedList.GroupDivider

Renders the group label/divider.

### RankedList.GroupItem

Container for an individual item.

### RankedList.GroupItemValue

Render prop component that receives the item and context.

| Prop | Type | Description |
|------|------|-------------|
| `children` | `(item: T, ctx: ItemContext) => ReactNode` | Render function |

### RankedList.GroupItemIndex

Renders the item's rank index.

### ItemContext

| Property | Type | Description |
|----------|------|-------------|
| `index` | `number` | Item's position in the list |
| `isLastInGroup` | `boolean` | Whether item is last in its group |
| `groupIndex` | `number` | Which group the item belongs to |
