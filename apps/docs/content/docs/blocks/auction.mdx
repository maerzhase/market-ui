## Overview

The **Auction** component is a complex UI block that combines multiple primitives to create a full-featured auction experience. It includes:

- **Bid Rankings** - RankedList-based display with preview slots
- **Live Countdown** - Real-time countdown to auction end
- **Stepped Bidding** - Tick-aware bid increments
- **Responsive Layout** - Adapts to different screen sizes
- **Keyboard Accessibility** - Full keyboard navigation support

## API Reference

### Auction.Provider

The root provider that supplies auction context to all sub-components.

| Prop | Type | Description |
|------|------|-------------|
| `auction` | `AuctionData` | The auction data object |
| `currentUser` | `User \| null` | Current authenticated user |
| `onBid` | `(amount: bigint) => Promise<void>` | Bid submission handler |

### AuctionData

```tsx
interface AuctionData {
  id: string;
  title: string;
  description?: string;
  imageUrl: string;
  startTime: Date;
  endTime: Date;
  editions: number;
  minBid: bigint;
  tickSize: bigint;
  decimals: number;
  bids: Bid[];
}
```

### Bid

```tsx
interface Bid {
  id: string;
  userId: string;
  userName: string;
  amount: bigint;
  timestamp: Date;
}
```
